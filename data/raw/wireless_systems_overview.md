# Wireless Systems Overview

This note summarizes several core topics in wireless communication:

- Beamforming uses antenna arrays to steer energy toward a desired direction.
- MIMO increases spatial multiplexing gain and link robustness.
- OFDM converts a frequency-selective channel into many narrowband subcarriers.
- Channel estimation helps the receiver recover fading and phase distortion.

The note is intended as a lightweight companion document for TeleRAG benchmark and demo queries.
